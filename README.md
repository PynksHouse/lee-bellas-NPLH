# Lee-Bella’s No Place Like Home CDC Website (GitHub Pages)

## What this is
A simple, fast, one-page website for:
- 24-hour childcare positioning
- programs/services
- transportation
- estimated rates
- employer partnerships
- founding families waitlist

## Setup
1) Create a GitHub repository (example: `leebellas-cdc-site`)
2) Upload these files:
   - index.html
   - styles.css
   - script.js
   - assets/logo.png
   - (optional) assets/social/og-image.png

## Turn on GitHub Pages
1) Go to your repo → Settings
2) Pages → "Build and deployment"
3) Source: Deploy from a branch
4) Branch: `main` and folder `/ (root)`
5) Save

GitHub will provide your live site URL.

## Customize
- Replace `YOUR_EMAIL_HERE` in:
  - index.html (mailto links)
  - script.js (WAITLIST_EMAIL)
- Replace `https://example.com/donate` with your real donation link
- Replace og-image at `assets/social/og-image.png` (1200x630 recommended)

## Notes
This site uses a mailto waitlist form (no server required).
If you want a Google Form instead, replace the form button link with your Google Form URL.
